# Estock Backend

## Overview

This is the backend API for the Estock project, built with Node.js, Express, and MSSQL.

## Features

- User Authentication
- Verify Email and Mobile
- Update User Phone
- Fetch Accounts and Dividends
- CRUD Operations for Users
